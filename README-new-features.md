## 🔧 技术实现细节

### 后端API（新增端点）

#### 资料库管理
- `GET /api/knowledge` - 获取所有知识文件
- `POST /api/knowledge` - 上传新知识文件
- `DELETE /api/knowledge/:filename` - 删除知识文件

#### 创意管理
- `GET /api/ideas` - 获取所有保存的创意
- `POST /api/ideas` - 保存新创意
- `PUT /api/ideas/:id` - 更新创意
- `DELETE /api/ideas/:id` - 删除创意

#### 核心功能（已增强）
- `POST /api/generate-ideas` - 现在会结合资料库内容生成创意
- `POST /api/generate-batch-scripts` - 批量生成脚本
- `GET /api/health` - 健康检查，显示新功能状态

## 🐛 故障排除

### 常见问题

#### 1. 服务器启动失败
**症状**：端口3002被占用
**解决**：
```bash
# 停止现有服务器
taskkill /F /IM node.exe

# 重新启动
node rok-ai-server-enhanced.js
```

#### 2. 资料库无法上传
**症状**：上传按钮无响应
**检查**：
- 服务器是否运行（访问 http://localhost:3002/api/health）
- 浏览器控制台是否有错误（F12 → Console）
- 网络连接是否正常

#### 3. 创意生成不包含资料库内容
**症状**：生成的创意与资料无关
**解决**：
- 确保资料库有相关内容
- 检查资料文件格式（纯文本，UTF-8编码）
- 重启服务器使新资料生效

#### 4. 页面无法访问
**症状**：404错误或空白页面
**解决**：
- 确保所有HTML文件在同一目录
- 使用相对路径访问
- 检查文件权限
